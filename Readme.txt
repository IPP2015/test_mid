simpan folder test ini pada computer atau laptop anda
install katalon studio
login katalon studio bila belum memiliki account anda dapat membuat nya terlebih dahulu di link ini https://katalon.com/sign-up
open katalon studio click Tools > update webdriver > 
bila sudah maka click file > click open folder > pilih folder yang telah didownload disimpan > click test cases > double click test no 21
setelah update webdriver dan open folder maka kembali ke folder yang didownload dan masuk ke folder SS dan copy pathnya dan update pada script path sesuai dengan petunjuk
bila sudah di update maka click run simbol >, bila sudah run maka masuk ke folder SS harus terdapat foto 18.JPG bila sudah selesai sudah
bila ingin report maka click test suites > click New Test Suite > click run simbol > bila sudah run maka buka folder report di folder 
yang telah disimpan buka folder yang terdapat tanggal atau angka dan buka terus sampai terdapat file pdf dan buka file tersebut
